# Data Visualization with Matplotlib

This module contains structured exercises and visualization projects using Matplotlib as part of my Data Science learning journey.

Key Areas Covered:
- Creating Line, Bar, and Scatter Plots
- Histograms and Pie Charts
- Plot Customization (titles, labels, legends, grids)
- Multiple plots and subplots
- Visualizing real-world datasets

These exercises help me understand how to transform data into meaningful visual insights.
